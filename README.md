# 🍳 クックブック (Cookbook)
日々の健康管理と体調に合わせた食事のための、自家製レシピ・メモを管理するリポジトリです。
このリポジトリは Public（公開）設定になっており、公開可能なレシピ情報を中心に格納しています。

## 📁 ディレクトリ構成
- [recipes/](recipes/) - カテゴリ分類されたレシピ集
  - [curries/](recipes/curries/) - スパイスカレーなどのカレーレシピ
  - [staples/](recipes/staples/) - 雑穀米などの主食レシピ
  - [side_dishes/](recipes/side_dishes/) - 常備菜や小鉢などの副菜レシピ
  - [fermented/](recipes/fermented/) - ヨーグルトなどの発酵食品レシピ
- [templates/](templates/) - 新しいレシピやメモを作成するためのテンプレート

## ⚠️ 運用上の注意 (Public リポジトリ)
- 個人情報（特定の予定、場所、健康状態のプライベートな数値など）はここには含めず、iCloud上のプライベートな Obsidian Vault（`Daily/`）にのみ書き込みます。
